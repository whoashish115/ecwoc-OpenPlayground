## ğŸ“– How to Use

1. Open `index.html` in your browser.
2. Click **Generate Palette** to create random colors.
3. Hover over a color to see its hex code.
4. Click on the hex code to copy it.
5. Double click on a color box to lock or unlock it.
6. Locked colors remain unchanged when generating new palettes.
