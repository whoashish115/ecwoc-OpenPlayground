How to Use
1. Select Chart Type
Choose from 6 different chart types in the "Chart Type" dropdown.

2. Choose Dataset
Select from 5 predefined datasets or use "Custom Data" to enter your own.

3. Customize Appearance
Adjust color scheme

Toggle grid, labels, and legend

Set animation speed

4. Work with Custom Data
Select "Custom Data" from the Dataset dropdown

Enter labels (comma-separated)

Enter values (comma-separated)

Click "Apply Custom Data"

5. Export Your Chart
Click "Export as PNG" to download the current visualization.
