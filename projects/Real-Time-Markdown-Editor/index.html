Basic Formatting

Keyboard Shortcuts

  • Ctrl/Cmd + S - Save document
  • Ctrl/Cmd + B - Bold text
  • Ctrl/Cmd + I - Italic text
  • Ctrl/Cmd + K - Insert link
  • Ctrl/Cmd + E - Toggle preview
