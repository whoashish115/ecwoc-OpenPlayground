How It Works

This UI uses CSS custom properties (variables) that are dynamically updated via JavaScript when you select an emotion. Each emotion has predefined values for colors, animations, typography, and spacing that create a distinct user experience.

Your selected emotion is saved to localStorage, so your preference will be remembered when you return.
